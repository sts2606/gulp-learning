# Gulp 5 Quick Start

Цей проєкт використовує Gulp 5 для автоматизації фронтенду:

- SCSS → CSS (мінімізація, autoprefixer, sourcemaps)
- Мінімізація та об’єднання JS
- Оптимізація зображень
- Live-сервер з BrowserSync

---

## Швидкий старт

1. Встановіть залежності:

```bash
npm install
```

2. Запустіть dev-сервер та слідкування за файлами:

```bash
gulp watch
```

Відкриється локальний сервер на http://localhost:3000
Зміни у SCSS, JS, HTML та зображеннях автоматично оновлюють сторінку
